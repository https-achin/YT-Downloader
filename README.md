# YouTube Video Downloader

This Python script allows users to download YouTube videos by selecting the desired quality. It features a simple menu interface and saves videos to a specific folder.

## Features
- Menu-driven interface
- Download videos from valid YouTube URLs
- Select video quality (480p, 720p, 1080p, etc.)
- Videos are saved to a specified folder
- User is notified of the download location

## Requirements
- Python 3.7+
- See `req.txt` for dependencies

## Usage
1. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```
2. Run the script:
   ```powershell
   python youtube_downloader.py
   ```
3. Follow the on-screen prompts to download your video.

## Notes
- Make sure you have a stable internet connection.
- The script uses `pytube` for downloading YouTube videos.
